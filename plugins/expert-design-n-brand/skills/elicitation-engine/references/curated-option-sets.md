# Curated Option Sets

Concrete palettes, typography pairings, voice examples, and archetype cards for use in recognition exercises. These are the actual options presented during discovery — not abstract descriptions, but specific, usable examples.

## How to Use This Reference

- **During visual discovery**: Present palettes and type pairings for recognition exercises
- **During voice discovery**: Use voice card sort examples
- **During personality discovery**: Use archetype quick cards for fast selection
- **For image generation**: Use palette hex values as input to image gen prompts

## Typography Pairings

Each pairing includes: heading + body combination, character description, specific Google Font recommendations, and CSS fallback stacks.

### Pairing 1: Classic Authority
**Character**: Traditional, trustworthy, editorial. The serious newspaper.
**Heading**: **Playfair Display** (serif) — high-contrast, elegant serifs
**Body**: **Source Sans 3** (sans-serif) — clean, readable, professional
**Fallback**: `"Playfair Display", Georgia, "Times New Roman", serif` / `"Source Sans 3", "Helvetica Neue", Arial, sans-serif`
**Best for**: Sophisticated + competent brands, editorial, finance, law
**CSS scale suggestion**: Major Third (1.250)

### Pairing 2: Modern Warmth
**Character**: Approachable, contemporary, friendly. Human but professional.
**Heading**: **DM Sans** (sans-serif) — geometric with subtle warmth
**Body**: **Inter** (sans-serif) — designed for screens, highly legible
**Fallback**: `"DM Sans", "Segoe UI", Roboto, sans-serif` / `Inter, "Segoe UI", Roboto, sans-serif`
**Best for**: Sincere + competent brands, SaaS, consumer apps, education
**CSS scale suggestion**: Perfect Fourth (1.333)

### Pairing 3: Bold Statement
**Character**: Strong, distinctive, unapologetic. Commands attention.
**Heading**: **Space Grotesk** (sans-serif) — geometric, techy, bold
**Body**: **IBM Plex Sans** (sans-serif) — engineered precision, humanist touches
**Fallback**: `"Space Grotesk", "Courier New", monospace` / `"IBM Plex Sans", "Helvetica Neue", Arial, sans-serif`
**Best for**: Exciting + competent brands, tech startups, developer tools, innovation
**CSS scale suggestion**: Major Third (1.250)

### Pairing 4: Elegant Simplicity
**Character**: Refined, airy, luxury. Whispers rather than shouts.
**Heading**: **Cormorant Garamond** (serif) — light, elegant, classical
**Body**: **Lato** (sans-serif) — warm, stable, semi-rounded
**Fallback**: `"Cormorant Garamond", Garamond, "Palatino Linotype", serif` / `Lato, "Trebuchet MS", sans-serif`
**Best for**: Sophisticated brands, fashion, beauty, boutique services
**CSS scale suggestion**: Minor Third (1.200)

### Pairing 5: Friendly Tech
**Character**: Casual, approachable, modern. The startup that doesn't take itself too seriously.
**Heading**: **Nunito** (sans-serif) — rounded terminals, playful but readable
**Body**: **Open Sans** (sans-serif) — neutral, highly legible, versatile
**Fallback**: `Nunito, "Segoe UI", Verdana, sans-serif` / `"Open Sans", "Helvetica Neue", Arial, sans-serif`
**Best for**: Sincere + exciting brands, consumer apps, education, community
**CSS scale suggestion**: Perfect Fourth (1.333)

### Pairing 6: Monospace + Humanist
**Character**: Technical, authentic, craft-oriented. Code meets design.
**Heading**: **JetBrains Mono** (monospace) — designed for code, distinctive
**Body**: **Atkinson Hyperlegible** (sans-serif) — maximum readability, unique letterforms
**Fallback**: `"JetBrains Mono", "Fira Code", "Courier New", monospace` / `"Atkinson Hyperlegible", Verdana, sans-serif`
**Best for**: Developer tools, tech companies, documentation, open-source
**CSS scale suggestion**: Major Third (1.250)

## Voice Card Sort Examples

Sample brand voice sentences for the card sort technique. Present 8-10 for the user to sort into "sounds like us" vs. "doesn't sound like us."

### Formal / Authoritative
1. "We've conducted extensive analysis to ensure this solution meets the highest standards of reliability."
2. "Our platform is trusted by industry leaders across six continents."

### Casual / Friendly
3. "Hey! We just shipped something cool and we can't wait for you to try it."
4. "Whoops — we hit a bump. Here's what happened and what we're doing about it."

### Bold / Provocative
5. "Everyone else is still using yesterday's playbook. We burned ours."
6. "If you're comfortable with the status quo, we're probably not for you."

### Warm / Caring
7. "We know change is hard. We'll walk you through every step."
8. "Your success matters more to us than our metrics. Seriously."

### Witty / Playful
9. "We take our work seriously. Ourselves? Not so much."
10. "Our competitors have bigger budgets. We have better taste."

### Technical / Precise
11. "Latency reduced by 47% with zero impact on throughput — here's the benchmark."
12. "Built on a zero-trust architecture with end-to-end encryption at rest and in transit."

### Minimalist / Understated
13. "Simple tools. Real results."
14. "Less noise. More signal."
